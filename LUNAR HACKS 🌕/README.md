# Branding Guidelines

## Colours
  Our logo uses pink (#c47ab3) and light blue (#3fa9f5) on a moon (see moon.png) with white (#ffffff) text on a navy (#1a1c26) background.

## Where to Use

| Logo             | Location                                                     |
| :--------------: | :----------------------------------------------------------: |
| Standard lockup  | Large horizontal space (banners, etc.)                       |
| Vertical lockup  | Portrait media (paper, etc.)                                 |
| Initials on moon | Small square or circular media (icons, avatars, etc.)        |
| Logotype on moon | Large circular media (stickers, etc.)                        |
